# 02 — Shapes and Transforms

This is where you build muscle memory with Manim's core vocabulary: **Mobjects**
(the things on screen) and **how to move them around**.

Think of it like learning GPIO before you can build a driver. Boring? Maybe.
Essential? Absolutely.

## What you'll learn

- Every common shape (Circle, Square, Rectangle, Line, Arrow, Polygon, etc.)
- Positioning: `shift()`, `move_to()`, `next_to()`, `align_to()`, `to_edge()`, `to_corner()`
- The coordinate system (default: x from -7 to 7, y from -4 to 4)
- `VGroup` — grouping objects so they move/scale together
- `Transform` vs `ReplacementTransform` — the difference matters
- `.arrange()` and `.arrange_in_grid()` for layout
- Building a chip pinout diagram from basic shapes
